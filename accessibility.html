PYRA Accessibility Statement

Pyramidion Cryptocurrency LLC intends for PYRA public pages and member workflows to be usable by people with disabilities, including people who use screen readers, keyboard navigation, browser zoom, or other assistive technology.

Prelaunch status: This public website is still being built. Accessibility reports are accepted now, while member workflows still require separate review before live onboarding.

Accessibility Goals

PYRA public pages should use semantic headings, descriptive link text, useful image alternatives, readable contrast, keyboard-accessible navigation, responsive layouts, browser zoom support, and form labels before production member activity opens.

Known Current Boundaries

The public landing page is informational and does not currently include live member forms, payment controls, account approval controls, KYC uploads, or blockchain transaction actions. Future app screens must receive separate accessibility review before production use.

Reporting Accessibility Problems

Report accessibility problems to damondanner@gmail.com. Include the page URL, the problem encountered, assistive technology used if any, browser/device information if available, and the action you were trying to complete. Do not include identity documents or private account credentials.

Manual Review

Automated accessibility checks are useful but not enough by themselves. PYRA should pair automated checks with manual keyboard, screen-reader, zoom, contrast, and form-label review before public member workflows go live.
